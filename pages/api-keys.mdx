## How to Create an API Key

<Steps>
  <Step title="Log in to Your Account">
    - Go to [ROOTKey's Platform](https://app.rootkey.ai/sign-in).
    - Enter your username and password.
    - Click on the **Log In** button.
  </Step>
  <Step title="Navigate to the Developer Tab">
    - After logging in, you are now on the **Dashboard**.
    - From the dashboard, locate and click on the **Developer** section in the sidebar.
    <img style={{ borderRadius: "0.5rem" }} src="https://www.media.notcybersec.com/docs/developer-tab-platform.png"/>
  </Step>
  <Step title="Create a New API Key">
    - In the **Developer Settings** section, click on the **Generate API Key** button.
    - Your new generated API Key will be available under the **Standard keys** section.
    <img style={{ borderRadius: "0.5rem" }} src="https://www.media.notcybersec.com/docs/developer-tab-platform-click.png"/>
  </Step>
  <Step title="Copy and Store Your API Key">
    - Once the API key is generated, it will be displayed on the screen.
    - Copy the API key to a secure location.
    <img style={{ borderRadius: "0.5rem" }} src="https://www.media.notcybersec.com/docs/developer-api-key-generated.png"/>
    - **Note:** To copy the key click on the _Other_ > **Copy API Key**.
  </Step>
  <Step title="Verify Your API Key">
    - To ensure your API key works correctly, go to any endpoint on the [API Reference](/api-reference/overview.mdx) section in the documentation.
    - Use the API key to make a request.
    - If the request is successful, your API key is ready to use.
  </Step>
</Steps>

## Best Practices for API Key Management

By following these steps, you can easily create and manage your ROOTKey API keys. 

<AccordionGroup>
    <Accordion title="Keep your API key secure" icon="file-shield">
        Never share your API key publicly or store it in insecure locations.
    </Accordion>
    <Accordion title="Regenerate your API key periodically" icon="arrows-spin">
        Regularly update your API key to maintain security.
    </Accordion>
    <Accordion title="Monitor API usage" icon="magnifying-glass">
        Keep track of how your API key is being used to detect any unusual activity.
    </Accordion>
</AccordionGroup>
